## ğŸ“¥ Getting Started

### Prerequisites

-   Node.js (v16+)
-   npm or yarn

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Deepak-999X/AI-Enabled-Large-Scale-Community-Mental-Health-Screening-Systems.git
    cd AI-Enabled-Large-Scale-Community-Mental-Health-Screening-Systems
    ```

2.  **Install Dependencies**:
    ```bash
    # Install client dependencies
    cd client
    npm install

    # Install server dependencies
    cd ../server
    npm install
    ```

3.  **Run the Application**:

    -   **Start the Backend**:
        ```bash
        cd server
        npm start
        ```
    -   **Start the Frontend**:
        ```bash
        cd client
        npm run dev
        ```

4.  **Access the App**:
    Open [http://localhost:5173](http://localhost:5173) in your browser.
